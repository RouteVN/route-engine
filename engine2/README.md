

Concepts

## Actions

Actions are performed on the engine to change state or cause side effects
All changes should be triggered by actions

## Visual State

This is a representation of everything on the stage. This has a Visual Novel language.

Visual State is generated from `steps`

## Variables State

This is a representation of all the variables in the VN

## Render State

This is a representation of everything on the stage, in the rendering language.

Visual State + Variables State = Render State
